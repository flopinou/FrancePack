## üá¨üáß English

Subway Builder modpack that adds several French cities (Paris, Lyon, Marseille, Lille, Toulouse, Rennes) to the game.

**Realistic demand data for all maps:** The game's generated traffic uses official French government data (INSEE and SIRENE database) for the distribution of population and jobs. This data is combined with attendance figures from the SNCF, the Ministry of Culture, and OpenStreetMap to simulate the flow towards train stations, airports, stadiums, and museums.

**Two installation methods are available:** üëâ [Script Method](#method-1-script-method) | üëâ [Map Manager Method](#method-2-map-manager-method)

* **The Script Method (Recommended):** Faster installation and provides the complete version of the mod. *(Note: If this method does not work for you, please fall back to the Map Manager Method).*
* **The Map Manager Method:** Recommended for a simpler installation process. While both methods provide the complete maps, this version lacks some secondary features, like translations and the "France" tab in the map selection menu.

### Method 1: Script Method

- Download the `FrancePack.zip` file from the [latest release](https://github.com/flopinou/FrancePack/releases/latest/download/FrancePack.zip).
- In Subway Builder, go to Settings > Mod Manager and click on "Open Mods Folder".
- Unzip (extract) the archive into the folder that opened.
    - *Note: The ZIP file contains the files directly (without a root folder), so make sure you extract them into a folder named `FrancePack` inside your `mods` folder (often done by default by extraction software).*
- Run `run_windows.bat` or `run_mac-linux.sh` (depending on your OS):
    - **On Windows:** Simply double-click on `run_windows.bat`.
    - **On macOS and Linux:** Open a Terminal, type `sh ` (with a space), drag and drop the `run_mac-linux.sh` file into the terminal window, and press Enter.
- Enable the mod inside Subway Builder, then click on "Reload Now".
- Click on New Game: you will find the maps in the **France** (or **Modded**) tab.

> **:warning: Important:** You must run the script (`.bat` or `.sh`) every time you want to play the map of a city in this modpack **before** launching the game (feel free to create a desktop shortcut for easier access!). Make sure the terminal window remains open in the background.

### Method 2: Map Manager Method

- Download the `FrancePack_MapManager.zip` file from the [latest release](https://github.com/flopinou/FrancePack/releases/latest/download/FrancePack_MapManager.zip).
- Unzip the downloaded ZIP file to get several individual `.zip` files (these are the maps you need to import).
- Follow the official Map Manager installation instructions here: [https://subwaybuildermodded.com/wiki/maps/map-installation-guide/](https://subwaybuildermodded.com/wiki/maps/map-installation-guide/). During Step 2, after clicking on "ADD A MAP", select all the individual `.zip` files you just extracted.
